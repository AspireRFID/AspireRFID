In order to build this project you need to:
- install JME WTK 2
- set the WTK_HOME environment variable
- Install Nokia 6131 SDK
- Copy Nokia 6131 SDK\lib\ext to WTK_HOME
- Copy Nokia 6131 SDK\lib\cldc11.zip to WTK_HOME\ext

Instead of doing the last two steps, you can also change this project's pom.xml
